# AI Transcriber 微信小程序

基于 UniApp 框架开发的微信小程序，支持 B站/YouTube 链接转录和本地文件上传。

## 功能特性

- ✅ 支持 B站 / YouTube 视频链接转录
- ✅ 支持本地音频/视频文件上传
- ✅ 实时查看转录进度
- ✅ AI 智能纠错和总结
- ✅ 任务历史记录

## 项目结构

```
AI_Transcriber_WX/
├── pages/
│   ├── index/          # 首页 - 创建转录任务
│   │   └── index.vue

│       └── tasks.vue
├── static/             # 静态资源（图标等）
├── App.vue             # 应用入口
├── main.js             # 主入口文件

└── manifest.json      # 应用配置
```

## 快速开始

### 方式一：使用 UniApp CLI（推荐）

1. **安装 Node.js**（如果还没有）
   - 下载地址：https://nodejs.org/

2. **创建 UniApp 项目**
   ```bash
   # 全局安装 vue-cli（如果还没有）
   npm install -g @vue/cli

   # 创建 UniApp 项目
   vue create -p dcloudio/uni-preset-vue my-ai-transcriber

   # 进入项目目录
   cd my-ai-transcriber

   # 安装依赖
   npm install
   ```

3. **复制小程序代码**
   - 将本目录下的所有文件复制到新创建的项目中
   - 覆盖原有的文件

4. **修改微信小程序 AppID**
   - 打开 `manifest.json`
   - 修改 `mp-weixin.appid` 为你的微信小程序 AppID
   - 或者在微信开发者工具中修改

5. **运行到微信开发者工具**
   ```bash
   npm run dev:mp-weixin
   ```

6. **在微信开发者工具中导入项目**
   - 打开微信开发者工具
   - 选择项目目录
   - 导入成功后会显示小程序预览

### 方式二：直接导入（不推荐）

1. 打开微信开发者工具
2. 选择「导入项目」
3. 选择本目录
4. 修改 AppID 为你自己的

## 配置说明

### API 地址配置

在代码中，API 地址默认指向：`http://124.221.77.205:8001/api`

如果后端部署在其他服务器，需要修改：
1. 打开 `pages/index/index.vue`
2. 修改 `API_BASE` 为实际的服务器地址
3. 同样修改 `pages/tasks/tasks.vue` 中的 `API_BASE`

### 微信小程序后台配置

如果需要让用户在外网访问你的后端 API，需要：

1. **登录微信小程序后台**（https://mp.weixin.qq.com）
2. **开发管理 → 开发设置**
3. **服务器域名配置**
   - 添加 `request 合法域名`：你的后端 API 地址（需要 HTTPS）
   - 添加 `uploadFile 合法域名**：同上

### 后端部署

后端服务需要部署在有公网 IP 的服务器上，并确保：
- 开启 HTTPS（微信小程序要求）
- 配置 CORS 允许跨域访问
- 服务器防火墙开放对应端口

## 常见问题

### 1. 微信小程序提示「request 合法域名校验失败」

这是因为微信小程序要求所有网络请求都必须配置在「服务器域名」中：

1. 登录微信小程序后台
2. 进入「开发管理」→「开发设置」
3. 在「服务器域名」中添加你的后端域名

**解决方案**：
- 开发测试时可以在微信开发者工具中勾选「不校验合法域名」
- 正式发布前必须配置合法域名

### 2. 后端 API 需要使用 HTTPS

微信小程序从 2017 年开始要求所有网络请求必须使用 HTTPS。

**解决方案**：
- 在服务器上配置 SSL 证书
- 使用 Nginx 或其他反向代理
- 或者使用云服务提供的 HTTPS 地址

### 3. 文件上传失败

微信小程序的文件上传有限制：
- 单个文件大小限制
- 文件类型限制

**解决方案**：
- 检查文件大小是否超过限制
- 确保文件格式正确
